# Geometry

This folder stores external geometry in the same spirit as the MOST examples:

```text
MOSTVolturnUS/
  IEA15MW_Blade.STL
  IEA15MW_Hub.STL
  IEA15MW_nacelle.STL
  IEA15MW_Tower.STL
  VolturnUS.STEP
```

Unity currently renders the turbine from STL files and uses a procedural semi-submersible approximation for the VolturnUS floater. The STEP file is kept here for CAD inspection or future mesh conversion.
