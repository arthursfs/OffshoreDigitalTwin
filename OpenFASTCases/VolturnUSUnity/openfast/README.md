# FAST/OpenFAST

Place FAST/OpenFAST comparison files and output traces here.

For reproducing pages 6 and 7 of the reference paper, export or convert channels for:

```text
time_s
fast_pitch_deg
fast_power_mw
fast_root_axial_force_kn
fast_root_axial_moment_mnm
```

The Unity HUD reads the merged MOST/FAST comparison CSV at:

```text
../most/outputs/sample_volturn_most_fast_comparison.csv
```
